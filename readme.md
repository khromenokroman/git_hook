# Git Hook Calculator

## Описание проекта
Простой проект калькулятора, написанный на C++. Проект использует библиотеку fmt для форматированного вывода и GoogleTest для тестирования.

## Требования
- C++20
- CMake (минимум 3.15)
- fmt
- GoogleTest

## Сборка проекта

### Клонирование репозитория
````bash 
git clone https://github.com/khromenokroman/git_hook
cd git_hook
````

### Сборка с помощью CMake

```bash 
mkdir -p build && cd build 
cmake .. 
make
```

## Запуск программы
После успешной сборки вы можете запустить программу:
````bash 
./src/git_hook
````

Программа выведет результаты простых математических операций:
````
Hello, I'm calculator! 
add(5, 2) = 7 
mul(5, 2) = 10
````

## Запуск тестов

Тесты можно запустить после сборки проекта:
```bash 
./tests/tests
```
