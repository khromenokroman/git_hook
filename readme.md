# Git Hook Calculator

## Описание проекта

Простой проект калькулятора, написанный на C++. Проект использует библиотеку fmt для форматированного вывода и
GoogleTest для тестирования.

## Требования

- C++20
- CMake (минимум 3.15)
- fmt
- GoogleTest

**Установка**

````bash
apt install build-essential libfmt-dev libgtest-dev 
````

### Git hooks

При разработке git-хуки выполняют некоторые проверки на работоспособность
и могут выполнять компоновку/форматирование при фиксации кода.
В текущем состоянии git-хуки выполняют следующую работу:

- Проверку форматирования кода (запускает `clang-format` для проверки форматирования)
- Проверку форматирования сообщения коммита (ожидается что в начале сообщения будут слова
  ***feat, fix, docs, style, refactor, perf, test, build, ci, chore, revert***)

**Для их работы требуется**

````bash
apt install clang-format
````

**Установка**

````bash
scripts/git-hooks/setup_hooks.sh
````

## Сборка проекта

### Клонирование репозитория

````bash 
git clone https://github.com/khromenokroman/git_hook
cd git_hook
````

### Сборка с помощью CMake

```bash 
mkdir -p build && cd build 
cmake .. 
make
```

## Запуск программы

После успешной сборки вы можете запустить программу:

````bash 
./src/git_hook
````

Программа выведет результаты простых математических операций:

````
Hello, I'm calculator! 
add(5, 2) = 7 
mul(5, 2) = 10
````

## Запуск тестов

Тесты можно запустить после сборки проекта:

```bash 
./tests/tests
```
